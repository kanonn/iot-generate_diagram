# 添加 Docker 仓库（如果 RHEL 的不行就用 CentOS 的）
sudo dnf config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

# 安装
sudo dnf install -y docker-ce docker-ce-cli containerd.io docker-compose-plugin --allowerasing

# 启动
sudo systemctl start docker
sudo systemctl enable docker

# 加入 docker 组（然后重新 SSH 登录）
sudo usermod -aG docker $(whoami)
```

### 整体流程
```
Phase 1  安装 Docker（RHEL 8 特有步骤）
Phase 2  安装 AWS CLI（可能已预装）
Phase 3  上传 JAR + Dockerfile 到 EC2
Phase 4  创建 Dockerfile（s3270 版本）
Phase 5  docker build
Phase 6  本地测试（可选）
Phase 7  登录 ECR → 打标签 → docker push
